## 技术栈

- HTML5 + CSS3
- GitHub Actions (CI/CD)
- GitHub Pages (部署)

## CI/CD 流程

1. 代码提交到 GitHub
2. GitHub Actions 自动触发构建
3. 自动部署到 GitHub Pages
